DESIGN SYSTEM:

Color Scheme:
- Primary: #2563eb (blue)
- Success: #16a34a (green) - normal/completed status
- Warning: #f59e0b (amber) - attention needed
- Danger: #dc2626 (red) - critical/error
- Info: #0ea5e9 (cyan) - information

Dark Theme:
- BG Primary: #0f172a (main background)
- BG Secondary: #1e293b (cards)
- BG Tertiary: #334155 (subtle backgrounds)
- Text Primary: #f1f5f9 (main text)
- Text Secondary: #cbd5e1 (secondary text)
- Border: #475569

Layout Dimensions:
- Sidebar expanded: 250px
- Sidebar collapsed: 64px
- Topbar height: 56px
- Responsive breakpoints: 768px (tablet), 480px (mobile)

UI COMPONENTS:

Buttons: .btn-primary, .btn-success, .btn-warning, .btn-danger, .btn-secondary
Badges: .badge-success, .badge-warning, .badge-danger, .badge-info
Alerts: .alert-success, .alert-warning, .alert-danger, .alert-info
Cards: .card with .card-header, .card-body, .card-footer
Tables: Standard HTML tables with dark theme
Forms: Input, textarea, select with focus states
Progress: .progress with .progress-bar variants
Grid: .grid with .grid-2, .grid-3, .grid-4 responsive layouts
